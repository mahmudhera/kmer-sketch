## FracMinHash
Test script name: `test_fracminhash.py`. In this script, we test that for all the fasta files in the data folder, we can generate the same sketch as generated by `sourmash sketch`

To run this test, stay in the parent directory (do not cd in the test directory) and run
```
python test/test_fracminhash.py 
```
